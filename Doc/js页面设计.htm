一、怎么使用JavaScript代码

1.直接插入的方式

现代浏览器都嵌入了JavaScript引擎，在想要的地方嵌入<script>…</script>的代码即可，它也是使用//进行注释的

2.使用脚本文件

JavaScript可以单独写在.js的文件中，然后在需要的地方用<script type=”text/JavaScript” src=”脚本全名.js”></script>进行引入

3.eclipse对JavaScript、html5的支持

下载Eclipse Java EE IDE for Web Developers

(1)使用空项目

建立一个空项目、空项目下再建立html文件；

点击项目->右键->Configure->Convert to JavaScript Project…

(2)使用javaee工程

这种做法的好处是html可以发布到局域网内，而不是使用浏览器打开本地文件的样式

Preferences->Workspace->Text file encoding选择UTF-8；

新建项目Dynamic Web Project；

选择Target runtime运行环境（默认是eclipse的J2EE Preview）、如果有别的需要再做其他设置；

在WebContent下写html文件，Run As->Run on Server

三、数据类型

1.数值

所有的数值都以浮点数的形式进行表示，用Math对象中的函数可以完成比较复杂的运算；.toString()或者与字符串对象做+号连接也可以将数值转换成字符串；浮点数运算可能会出现一些问题，比如0.1+0.2!=0.3，但是整数的运算是精确的、可以先转换成整数再运算

2.字符串

用英文单引号或双引号括起来的字符序列，如果一行写不完用+号进行连接；汉字等unicode的字符满足str.charCodeAt(i)>255，unicode转换成十进制后再转换为字符String.fromCharCode(十进制数)

a.元字符

b.可能出现的形式

^

在左中括号外表示开头

$

表示结尾，比如/n$/表示n结尾

/[字符0…字符n]/

/[字符0…字符n]/g

/[字符0…字符n]/i

之间是或的关系，比较特殊的是[a-z]、[0-9]，g表示执行多次替换、i表示忽略大小写、不写表示执行一次替换

/[^字符0…字符n]选择

非字符0-字符n

/表达式|表达式/选择

之间也是或的关系

/…(?:表达式)…/选择

非捕获性分组，不会在“$数字”中写数据

/...表达式0(?=表达式1).../选择

表达式0后面恰好是表达式1

/...表达式0(?!表达式1).../选择

表达式0后面恰好不是表达式1

/…表达式*表达式…/g

贪婪匹配，“先吞下整个字符串，不匹配再一个个的去掉最后的字符”得到结果是尽可能长的数据

/…表达式*?表达式…/g

惰性匹配，“先吞下第一个字符，不匹配再吞下一个”得到结果是尽可能短的多个数据

       function f(a, b) {

              return a-b;

       }

       arr.sort(f);

       alert(arr);

</script>

①window对象

即顶层对象、根节点，也是BOM的核心、代表浏览器窗口的一个实例；self、parent、top等顶层对象也是window对象，在标签<body>或<frameset>出现时被创建；window.innerWidth表示浏览器窗口的宽度、window.innerHeight表示浏览器窗口的高度

a.通过“window.”定义的变量可以被delete，而被定义的变量不能被delete；可以用“window.变量”安全的判断变量是否被定义过

<script type="text/JavaScript">

       var a = "a";

       window.b = "b";

       alert(window.a);

       alert(b);

       //true，可以被delete

       alert(delete window.b);

       //false，不能被delete

       alert(delete a);

       alert(a);

       //不能再被访问到

       alert(b);

</script>

2.JavaScript事件绑定

(1)静态绑定

使用标签中的onclick=”某个函数或则javascript代码”的方式，这种方式用起来比较简单灵活、缺点是破坏了HTML结构

(2)动态绑定

注册事件处理函数，第一个参数去掉各种事件中的“on”；移动端使用touchstart、touchmove、touchend；仅在要用到的时候才调用代码，兼容性较好

对象. addEventListener("类型", function() {代码}或者一个函数, false)

(3)事件处理函数

注销事件处理函数“对象. removeEventListener("类型", function(){代码}或者一个函数, false);”；JavaScript中的颜色可以使用”rgb(,,)”来表示，JavaScript有时会调用一些带“-”的属性，比如“text-align”，应该用驼峰法表示“obj.style.textAlign”

for (var i = 0; i < arr.length; ++i) {

七、JavaScript进阶Ajax

和一般的JavaScript代码不同，Ajax只有部署在站点上才会工作；支持读数据 ，但是不能写数据

1.XMLHttpRequest对象

(1)常用方法

new XMLHttpRequest()

XMLHttpRequest的构造函数

abort()

取消当前请求

getAllResponseHeader()

获取所有响应的HTTP头信息

getResponseHeader()

获取指定的HTTP头信息

open(“get或post”,”带协议的url地址”，布尔值常用false)

与服务器建立连接，布尔值如果是ture表示异步、否则同步

send(“”)

发送一个HTTP请求并接收返回的数据，参数可以为null

setRequestHeader()

指定某个HTTP头信息

(2)常用属性

onreadystatechange

readyState改变时发生作用的事件处理句柄，后面接一个函数对象即可

readyState

当前请求的状态，一般和status一起使用：

0未调用open()，未初始化；

1未调用send()，初始化；

2已发送数据，但服务器状态未知；

3已接收部分数据；

4数据接收完毕。

responseBody

回应消息正文以unsigned byte数组的形式返回

responseStream

回应消息以ADO Steam对象的形式返回

responseText

响应消息以字符串的形式返回

responseXML

响应消息以XML的形式返回

status

当前请求的http状态码：

200客户端请求成功；

400请求的代码有误；

403服务器拒绝该请求；

404服务器找不到对应的资源。

statusText

当前请求的响应行状态

2.post请求

也可以换成get请求

var xmlHttp = new XMLHttpRequest();

xmlHttp.open("post", "server.do", false);

//xmlHttp.setRequestHeader("Content-type","aplication/x-www-form-urlencoded");

xmlHttp.send(null);

console.log(xmlHttp.responseText);

3.解析xml

var xmlHttp = new XMLHttpRequest();

var url = "server.xml";

xmlHttp.open("GET", url, true);

xmlHttp.onreadystatechange = function() {

       if (xmlHttp.readyState == 4 && xmlHttp.status == 200) {

              var info = xmlHttp.responseXML;

              alert(info.getElementsByTagName("big")[0].firstChild.data);

       }

}

xmlHttp.send(null);

 

八、数据传输和存储

1.使用cookie

cookie在站里面才会有效，默认是同级页面、子页面可以访问到cookie（添加“;path=/”可以扩展到所有目录）

//设置cookie的有效时间，也可以使用setHours()、getHours()设置为小时，setMinutes()、getMinutes()设置为分钟

var nDate = new Date();

nDate.setDate(nDate.getDate() + 30);

//直接接在后面expires才会有效

document.cookie = ",username=Darren,key=123456;expires="

              + nDate.toGMTString();

//获取cookie

var getC, c_arr0, c_arr1, c_res;

getC = unescape(document.cookie.substring(0));

if (getC != "") {

       c_arr0 = getC.split(",");

       for (var i = 0; i < c_arr0.length; ++i) {

              c_arr1 = c_arr0[i].split("=");

              //key是键，要获取的是值

              if (c_arr1[0] == "key") {

                     c_res = c_arr1[1];

              }

       }

}
